# LG TV Remote — Get Your APK

## Quick start (3 steps)

### Step 1 — Install Node.js
Download from: https://nodejs.org
Pick the **LTS** version. Run the installer and accept all defaults.

### Step 2 — Double-click BUILD.bat
It will:
- Install everything automatically
- Ask you to log in to a free Expo account (https://expo.dev — sign up first)
- Build your APK in the cloud (~10-15 minutes)
- Give you a **download link**

### Step 3 — Install on your phone
1. Open the download link on your phone (or copy it to phone browser)
2. Tap the `.apk` file to download it
3. Tap the downloaded file to install it
4. If Android asks "Allow install from unknown sources" → tap **Allow** → Install

---

## Using the app

1. Open the app → tap **Set TV IP Address**
2. Find your TV's IP: on the TV go to
   **Settings → Network → Wi-Fi Connection → Advanced Wi-Fi Settings**
3. Type the IP (like `192.168.1.105`) → tap **Connect**
4. **A popup will appear on your TV screen** — press **Allow**
5. Remote is ready!

The IP is saved — next time it connects automatically.

---

## Troubleshooting

| Problem | Fix |
|---------|-----|
| "Could not reach TV" | Make sure TV is on and both on same WiFi |
| No popup on TV | TV Settings → General → Allow Mobile Device Access → ON |
| Arrows don't work | Disconnect and reconnect — input socket needs a moment |
| IP changed after router restart | Open app → tap "Change TV IP" → enter new IP |
